📸 Image Gallery

A responsive image gallery website built using HTML, CSS, and JavaScript. This project displays a collection of images organized into different categories and allows users to filter images by selecting category buttons.

🚀 Features

✅ Responsive design that works across different screen sizes
✅ Image gallery layout with multiple images
✅ Category filtering buttons (All, Mountains, Beaches, Flowers, Forests)
✅ Interactive image display using JavaScript
✅ Hover effects for images
✅ Clean and user-friendly interface
🛠️ Technologies Used
HTML5 – Used to create the structure of the webpage
CSS3 – Used for styling, layout, responsiveness, and hover effects
JavaScript – Used to add functionality such as image filtering
🎯 Project Purpose
This project was created as part of the CodeAlpha Internship Program to practice and improve front-end web development skills using HTML, CSS, and JavaScript.

💡 What I Learned

Through this project, I practiced:

Creating a responsive webpage layout
Styling images using CSS
Organizing files and project structure
Using JavaScript to manipulate webpage elements
Implementing category-based image filtering
